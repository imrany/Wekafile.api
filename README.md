# Fileshare API
Manage and share your files with Fileshare.


# Contribution 
Fork the repo then clone
```bash
git clone https://github.com/imrany/Fileshare.server
```
then 
```bash
cd Fileshare.server
```
Engine configurations,
ensure you have
`node v14.19.1` 
and 
`npm v6.14.16`
then
```bash
npm i
```

Then Run
```bash 
docker-compose up -d
```
